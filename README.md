# Arch Linux Dotfiles

  ### Bear in mind that this enviroment utilizes:
  - **Window Manager:** BSPWM
  - **Application Launcher:** Rofi
  - **Status Bar:** Polybar
  - **Terminal Emulator:** Kitty
  - **Text Editor:** Neovim

<details>

  ### For this environment to function properly, it is necessary to install the following packages:
  - xorg-server 
  - xorg-xsetroot
  - xorg-xrandr
  - xorg-xinit
  - flameshot
  - pavucontrol 
  - pulseaudio
  - htop
  - nautilus
  - discord
  - obs-studio
  - libreoffice
  - p7zip
  - zip
  - unzip
  - pacman-contrib
  - neofetch
  - xf86-video-qxl
  - python
  - openssh
  - bspwm
  - sxhkd
  - rofi
  - nitrogen
  - picom
  - kitty
  - chromium
  - ttf-hack-nerd
  - firefox
  - zsh
  - git
  - bat
  - lsd
  - neovim
  - ranger
  - npm

  ### You can copy the following command to install all the packages:
  ```
  sudo pacman -S --noconfirm xorg-server xorg-xsetroot xorg-xrandr xorg-xinit flameshot pavucontrol pulseaudio htop nautilus discord obs-studio libreoffice p7zip zip unzip pacman-contrib neofetch xf86-video-qxl python openssh bspwm sxhkd rofi nitrogen picom kitty chromium ttf-hack-nerd firefox zsh git bat lsd neovim ranger npm
  ```
